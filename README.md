# オマーキ — Root-Yaru
 
> *A dark, elegant Base16 theme for Omarchy. Where Tokyo nights meet terminal precision.*
 
---
 
## About
 
**Root-Yaru** is a handcrafted Base16 color theme for [Omarchy](https://github.com/basecamp/omarchy) (Arch Linux + Hyprland). Inspired by Japanese aesthetics and sci-fi culture, it blends the moody stillness of a Tokyo night with the cold precision of a terminal interface.
 
Built around a deep charcoal void and a striking sakura pink accent — clean, minimal, and quietly beautiful.
